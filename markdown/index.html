dokieli with Markdown and Smartdown

dokie.li is a clientside editor for decentralised article publishing, annotations and social interactions.

This site is the published output of my fork of the original dokieli repo

Markdown and Smartdown Extensions to dokieli

My basic goal was to be able to leverage one of dokieli's main features, the self-replicating editable document, and enable Markdown and Smartdown to be authored within this environment.

I achieved this by enabling a dokieli view of type Markdown to be added to a document, and this view could then be rendered and edited via a Markdown editor that I added to the dokieli codebase. Additionally, I added commands to transform selected dokieli text into Markdown, and vice versa.
